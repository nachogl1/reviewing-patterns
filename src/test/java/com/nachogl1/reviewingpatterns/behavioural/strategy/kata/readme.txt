**Information about the kata:**
*This kata aims for the developer to use Strategy Pattern, this is a behavioural pattern
which allows you to select an algorithm/strategy among different ones to be used at runtime.
Another example of use Strategy Pattern is implementing different payment methods.*

# Cool Calculator

You need to design a system which will receive two numbers and type of operation to perform at runtime. The three
operations will be

- Addition
- Subtraction
- Multiplication

After choosing the operation at runtime, that will be applied to the numbers that were provided.

Test the following statements:

- 1+1 = 2
- 2-1 = 1
- 2x2 = 4
